我曾使用 Java、Spring Boot、Nest.js 构建后台系统，但始终存在两个痛点：性能浪费 & 架构维护难。

于是我开始构建一个 Rust 全栈模板 —— `rustzen-admin`，它拥有：

- 🌟 后端：Axum + sqlx + RBAC 权限 + JWT
- 🎨 前端：Vite + React + Tailwind + Zustand
- 🚀 部署：Rust 二进制，易于私有部署
